Dataspace Simulatie

Deze simulatie laat zien hoe een Dataspace werkt: een netwerk waarin Data Providers hun gegevens beschikbaar stellen via LDES en notificaties sturen via LDN naar een Registry (de kern van de dataspace).

Wat gebeurt er?

  • Provider → LDES: Provider publiceert data op een Linked Data Event Stream. Dit is de feitelijke dataset die je kunt ophalen.
  • Provider → LDN → Registry: Provider stuurt een Linked Data Notification naar de registry om te melden dat er nieuwe data beschikbaar is. De registry haalt dan de data via LDES op.
  • Registry (Dataspace): Ontvangt de notificatie, haalt LDES-data op en bouwt een gezamenlijke catalogus van alle metadata.
